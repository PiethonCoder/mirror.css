Mirror

.mirror is a class that will reflect the background image or color assigned to it.
A solid background color is needed for the effect to work. copy :root {--shade: yourColorHere;} into your css, and change 'yourColorHere' to your background color

Not working? Heres some tips.
  • make sure you changed :root --shade to your color
  • the effect only works with a solid background color
  • if there is a slim visible line, ajust the elements width slightly
    *this is a browser rendering issue

Mirror Text

.mirror-text is a class that will give you a reflection of your text. Add the .miror-text class to any h1, p, span ect. tag that has text in it, and give the element the data-content attribute, set equal to the text that you want to mirror. <h1 class="mirror-text" data-content="hello world">hello world</h1>

Glass

.glass is a simple class that makes a element have a subtle design.

you can add the mirror-text in combination with the glass class for an interesting design.

Foggy Glass

.foggy-glass will give you a blurry background for the image of your choosing. Give a element the .foggy-glass and a background-image.
<div style="background-image:url(img.jpg);background-position:center;"> <div class="foggy-glass" style="background-image:url(img.jpg);"></div> </div>

if your background element and the blurry element have the same background image, you get this kind of effect.
